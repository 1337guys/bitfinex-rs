# bitfinex-rs
Rust Library for the [Bitfinex API](https://bitfinex.readme.io/v2/docs/getting-started)

# Usage

Add this to your Cargo.toml

```toml
[dependencies]
bitfinex = { git = "https://github.com/wisespace-io/bitfinex-rs.git" }
```
